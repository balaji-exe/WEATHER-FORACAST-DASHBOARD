# Weather Forecast Dashboard (Power BI)

## Overview
This project is a Power BI dashboard that analyzes weather data including temperature, humidity, and rainfall patterns.

## Features
- KPI card for average temperature
- Temperature trends over time (line chart)
- Rainfall comparison across cities (column chart)
- Weather condition distribution (pie chart)
- Temperature vs humidity analysis (scatter chart)

## Tools Used
- Power BI
- Excel

## Dataset
The dataset used is included in this repository.

- Temperature trends vary across cities
